      3 FORMAS DE DECLARAR VARIÁVEIS

    * 1 normal
        * int idade = 30;
        * float sal = 123.34f;
        * letra char = 'G';
        * casado booleano = falso;
        *
    * 2 TYPECAST
        * int idade = (int) 30; 
        * float sal = (float) 123,34;
        * letra char = (char) 'G';
        * boolean casado = (boolean) false;
        *
    * 3 CLASSE WRAPPER (são classes, não é variavél e sim um objeto)
        *
        * Integer idade = Inteiro novo (30); 
        * Float sal = novo Float (123,34)
        * Letra do caractere = novo caractere ('G');
        * Booleano casado = novo Booleano (falso);
        
        
        FAMILIA  |   TIPO    |  CLASSE   |  TAMANHO  |  EXEMPLO
                 | PRIMITIVO | INVÓLUCRO |
        Lógico   |  boolean  |  Boolean  |   1 bit   |   true
        
        Literais |   char    | Character |   1 byte  |   'A'
        Literais |    -      |  String   |1 byte/cada|  "JAVA"
        
        Inteiros |   byte    |   Byte    |   1 byte  |   127
        Inteiros |   short   |   Short   |   2 bytes |  32 767
        Inteiros |    int    |  Integer  |   4 bytes | 2 147 483
        Inteiros |   long    |   Long    |   8 bytes |2 (elevado 63)
        
        Reais    |   float   |   Float   |   4 bytes |3.4e (elevado +38)
        Reais    |  double   |  Double   |   8 bytes |1.8e (elevado +308)
